## What Shipped (1.6.0 + 1.6.1)

### Calm UI Redesign (1.6.0)
- Header collapsed to single row with status, history, new chat + overflow menu (Checkpoints, Memory, Context, Extensions, Export, Diagnostics, settings, help).
- Composer: calm default (attach, permission mode, context meter, Send) + advanced row (model picker, Local/Search, sketch, ACP badge) behind persistent toggle.
- Always-visible context window meter next to Send: colour-coded ring (green/amber@80%/red@95%), token breakdown on hover/click to Context Dashboard.
- Welcome simplified to logo + tagline + prompt chips + single setup link.
- Thinking indicator: one message per turn (no rotation). Retry on hover.
- Gemini branding: activity bar icon, welcome, thinking indicator now use Gemini logo (Veles retained only for 1.5.x internal builds). Note in CHANGELOG.

- Warning/stall notices: stronger text contrast, comfortable padding.
- Composer controls share consistent 32px height rhythm for vertical centering.
- Package: `displayName` = "CalmUI for Gemini CLI", `repository` set to public `https://github.com/samshennan/calmui-for-gemini-cli.git` (duplicate private key removed in follow-up commit).
- README: Google non-affiliation + trademark disclaimer added; visual tour mirrored from public (new assets in `docs/`: hero-welcome.png, advanced-controls.png, permission-modes.png, before-redesign.png).
- VSIX: `gemini-cli-calmui-1.6.1.vsix` built.
